## [0.0.2] - 2025-08-07 

## Added

  - XSS detection
  - Secrets detection 
  - Command Injection
  - Insecure `eval()` / `Function()` Usage
  - Insecure Deserialization
  - Weak Cryptography
  - Unrestricted File Upload
  - Insecure Cookie Flags
